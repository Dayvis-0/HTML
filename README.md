# HTML Learning Repository

Repositorio de aprendizaje de HTML con ejemplos prácticos y ejercicios.

## Estructura

- **0. Teoria Google/** - Conceptos básicos de HTML
- **1. Teoria Soy dalto/** - Teoría y ejemplos de etiquetas HTML
- **1. Practica Soy dalto/** - Ejercicios prácticos
- **2. Teoria MDN/** - Documentación MDN
- **3. Uni/** - Ejercicios universitarios
- **4. Roadmap/** - Roadmap de aprendizaje con ejemplos progresivos

## Contenido

Ejemplos de etiquetas HTML, formularios, tablas, multimedia, listas, atributos estándar y más.

## Temas cubiertos

- Etiquetas básicas (headings, párrafos, enlaces)
- Atributos estándar (id, class, style, data-*)
- Listas (ordenadas, desordenadas, de definición)
- Tablas y captions
- Multimedia (imágenes, audio, video)
- Formularios y elementos de entrada
- Estructura semántica (nav, section, article, aside)
- Metadatos y configuración del documento
