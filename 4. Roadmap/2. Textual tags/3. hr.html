Horizontal rule

The <hr> HTML element represents a thematic break between paragraph-level elements: for example, a change of scene in a story, or a shift of topic within a section.

Historically, the <hr> element has always been presented as a horizontal rule or line. While it may still be displayed as a horizontal rule in visual browsers, this element is now defined in semantic terms, rather than presentational terms.

Example

§1: The first rule of Fight Club is: You do not tal about Fight Club.

§: The second rule of Fight Club is: Always bring cupcakes
