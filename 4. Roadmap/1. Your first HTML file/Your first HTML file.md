# 1. Your First HTML File

This folder contains fundamental practical examples to understand basic HTML concepts.

## Contents

### 1. Tags and attributes (`1. Tags and attributes.html`)
- Demonstrates how to use basic HTML tags
- Shows common attributes like `class`, `id`, `href`
- Examples of element structure

### 2. Case Insensitivity (`2. Case Insensitivity.html`)
- Explores that HTML is not case-sensitive
- Compares `<DIV>` vs `<div>` vs `<Div>`
- Demonstrates language flexibility

### 3. HTML entities (`3. HTML entities.html`)
- Shows how to use special characters
- Examples of `&lt;`, `&gt;`, `&amp;`, `&quot;`
- Demonstrates when to use entities vs direct characters

### 4. Whitespace (`4. Whitespace.html`)
- Explains how HTML handles whitespace
- Demonstrates collapsing of multiple spaces
- Shows differences between spaces and `&nbsp;` entities

## Learning Objectives

Understand essential HTML fundamentals:
- Basic document structure
- Proper use of tags and attributes
- Special character handling
- Whitespace behavior

## Getting Started

1. Open each HTML file in your browser
2. View source code (Ctrl+U)
3. Use "Inspect Element" (F12) to see processed DOM
4. Compare differences between original and rendered code

## 💡 Pro Tips

1. **Always start with DOCTYPE**: Never skip the DOCTYPE declaration
2. **Set UTF-8 charset**: Prevents character encoding issues
3. **Use lowercase tags**: While HTML is case-insensitive, lowercase is the standard
4. **Close all tags**: Missing closing tags can cause rendering issues
5. **Use HTML entities**: For special characters like `<`, `>`, `&`

## 📊 Quick Comparison

| Feature | Description | Best Practice |
|---------|-------------|----------------|
| Tags | HTML elements | Always close them |
| Attributes | Element modifiers | Use quotes around values |
| Case Sensitivity | HTML is case-insensitive | Use lowercase |
| Whitespace | Collapses in rendering | Use entities for exact spacing |
| Entities | Special character codes | Use `&lt;`, `&gt;`, `&amp;` |
